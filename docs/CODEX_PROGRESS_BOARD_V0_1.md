## 4. 后续任务入口

下一阶段只接受以下任务：

```text
1. 连续陪伴与低压回访：围绕轻记忆、开场回访、换题放下、拍图作品宽泛延续。
2. 主动回应与等待感优化：围绕轮流说话、等待状态、首段语音体感、打断停止和自然收回。
3. 常驻轻共创：围绕故事接龙、拍图起名、拍图讲一句小故事。
4. App 界面重设计：围绕主控生图提示语、ChatGPT image-2 方向图、主控评审、Android 落地和真机回看。
5. 真机体验小修：只修孩子或家长真实感受到的问题。
```

每个任务必须写清楚：

```text
当前孩子感受到什么问题；
改完后孩子能直接感受到什么变化；
修改范围；
禁止范围；
如何确认不是原地打转；
远程主分支提交编号。
```

## 5. 真机体验检查重点

```text
1. 打开儿童页，小白狐是否第一眼可见；
2. 横屏下小白狐、消息区、语音按钮是否互相挤压；
3. 文字是否先出现，首段语音是否仍慢；
4. 停一下是否立刻停止旧语音；
5. 静音后是否保留文字但不自动朗读；
6. 小白狐说完后等待孩子继续说是否自然；
7. 拍照、相册、取消、失败、成功是否顺；
8. 普通图片是否进入创作分享，题目图片是否进入学习帮助；
9. 轻记忆是否偶尔出现，不频繁、不追踪、不拉回孩子不想聊的话题；
10. 常驻轻共创是否像陪伴，不像任务、游戏化或学习平台；
11. 界面重设计是否让孩子更容易理解小白狐在听、想、说、看图、等待或收尾；
12. 日志是否仍避免孩子原文、图片内容、音频内容和完整接口响应。
```
