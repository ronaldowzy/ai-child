## 主控会话验收清单

```text
1. 是否遵守 AGENTS.md。
2. 是否绕过 ModelRegistry、PromptManager、SafetyEngine、SceneOrchestrator 等架构边界。
3. 是否新增或更新测试。
4. 是否运行测试并报告真实结果。
5. 是否引入真实 secret 或真实外部模型调用。
6. 是否保存不必要的儿童原始数据。
7. 是否有越界修改或隐藏风险。
8. 是否把已知环境坑误报为新阻塞。
```
