## 2. 关于拍图状态的判断

当前关键口径已经修正：

```text
上传中：正在给小白狐看看；
成功/处理中：小白狐正在看；
失败：这张图还没给小白狐看到。
```

其中 `ImageProcessing -> MascotState.ImageViewing` 是正确的。

理由：

```text
当图片已经进入后端处理阶段，孩子应该感受到“小白狐正在看”，而不是普通“正在想”。
```

需要继续保留的边界：

```text
1. 图片失败不得显示 image_viewing；
2. 图片失败不得显示 co_create；
3. 本地刚选中但尚未上传成功，不得说“小白狐已经看到”；
4. 失败时不得出现“起个名字 / 讲一句小故事”；
5. 成功后如有共创入口，同一张图最多一个。
```
