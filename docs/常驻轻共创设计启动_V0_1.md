## 1. 第 3 轮方向

第 3 轮名称：

```text
常驻轻共创
```

核心问题：

```text
小白狐目前能回应、能等待、能轻轻记得，但还缺少孩子愿意反复回来继续的小小共同内容。
```

本轮不是做游戏中心，不是做任务系统，不是做课程，不是做作品库，而是让小白狐具备常驻、低压、可随时停止的轻共创能力。

孩子应该感到：

```text
我可以和小白狐一起接一句故事；
我可以给图里的东西起个名字；
我可以说一点点，不用完成任务；
我不想继续时，小白狐会停下。
```

## 三、本轮只设计 3 个核心场景

### 场景 1：故事接龙

小白狐可以在普通对话中轻轻发起“你一句，我一句”的故事接龙。

重点判断：

什么时候适合发起；
怎样发起才不像任务；
孩子接一句后，小白狐如何接一句；
最多推进几轮；
孩子不接时如何放下；
怎样避免变成游戏、挑战或课程。

### 场景 2：拍图起名

孩子拍了画、手工、积木、玩具、泥塑、折纸等普通创作型作品后，小白狐可以邀请孩子给它起一个名字。

重点判断：

哪些图片适合起名；
哪些图片必须避让；
怎样不复述具体隐私细节；
小白狐能不能给 1-2 个名字方向；
孩子不选、不接、换题时如何放下。

### 场景 3：拍图讲一句小故事

孩子拍图后，小白狐可以围绕“作品”或“今天看到的东西”轻轻邀请孩子讲一句小故事。

重点判断：

如何只讲一句，不变成写作文；
如何避免变成学习任务；
小白狐如何接孩子一句；
如何自然收尾；
如何避免作品库、成长记录或长期追踪感。
